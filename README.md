# Rock-Paper-Scissors Image Classification using YOLO

An image classification system that recognizes **Rock**, **Paper**, **Scissors**, and **Empty** hand gestures using Ultralytics YOLO and transfer learning.

## Results
- ✅ 100% validation accuracy
- ✅ Real-time inference with confidence scores
- ✅ Tested on unseen images

## Tech Stack
- Python
- Ultralytics YOLOv8
- Roboflow (dataset preparation)
- Google Colab
- Transfer Learning

## Project Structure
- `rock_paper_scissors_yolo.ipynb` — Full training and inference pipeline
- `data.yaml` — Dataset configuration
- `model.pt` — Trained model weights (see Google Drive link below if not in repo)

## Dataset
Prepared using [Roboflow](https://roboflow.com). Contains labeled images of Rock, Paper, Scissors, and Empty hand gestures.

## How to Run
1. Open the notebook in Google Colab
2. Upload `data.yaml` and your dataset
3. Run all cells sequentially
